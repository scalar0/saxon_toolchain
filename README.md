# Saxon Toolchain Bazel Rules

This standalone Bazel module provides a Saxon-HE execution toolchain and reusable XSLT transform rule for XML/XSLT-native Bazel pipelines.

## Contents

- `//:saxon_cli` - Java executable target for `net.sf.saxon.Transform`.
- `//:defs.bzl` - public Starlark API.
- `//:xslt_rules.bzl` - reusable `xslt_transform(...)` implementation.
- `//toolchain:toolchain_type` - Bazel toolchain type for Saxon consumers.
- `//toolchain:saxon_toolchain` - default registered toolchain.
- `SaxonToolchainInfo` - provider exposing the executable and pinned version.

## Version

Saxon-HE is pinned to Maven coordinate `net.sf.saxon:Saxon-HE:12.9` through the BCR `rules_jvm_external` module. Runtime dependencies are resolved by the dedicated `saxon_maven` repository generated by the module extension.

## Validation

From the root module:

```bash
bazel test @saxon_toolchain//tests:all
```

The tests cover:

- analysis-time toolchain provider resolution;
- Saxon CLI startup;
- execution of a minimal XSLT transform through the public `xslt_transform(...)` rule.
