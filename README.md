# DCSE Saxon-HE Bazel Toolchain

This standalone Bazel module provides the Saxon-HE execution toolchain used by DCSE's XML/XSLT-native document pipeline.

## Contents

- `//:saxon_he_cli` - Java executable target for `net.sf.saxon.Transform`.
- `//toolchain:toolchain_type` - Bazel toolchain type for Saxon-HE consumers.
- `//toolchain:saxon_he_toolchain` - default registered toolchain.
- `SaxonHeToolchainInfo` - provider exposing the executable and pinned version.

## Version

Saxon-HE is pinned to Maven coordinate `net.sf.saxon:Saxon-HE:12.9` through `rules_jvm_external`. Runtime dependencies are resolved by the dedicated `saxon_maven` repository generated by the module extension.

## Validation

From the DCSE root module:

```bash
bazel test @saxon_he//tests:all
```

The tests cover:

- analysis-time toolchain provider resolution;
- Saxon CLI startup;
- execution of a minimal XSLT transform through the toolchain.
